Para compilar o projeto, e necessario ter instalado:
- g++
- mingw32-make

Para compilar:
mingw32-make

Para executar os testes:
mingw32-make test

Para gerar cobertura:
mingw32-make gcov

Observacoes:
O arquivo cpplint.py fornecido apresentou incompatibilidade com a versao atual do Python no ambiente Windows.

O comando cppcheck nao estava disponivel no ambiente Windows utilizado.

O comando valgrind nao e suportado nativamente no Windows.

Mesmo assim, o projeto foi compilado, executado e testado normalmente, incluindo a geracao de cobertura com gcov.
